An is a method which can declaratively substitute a call to an method with a call to itself at compile time. This substitution occurs by having the interceptor declare the source locations of the calls that it intercepts. This provides a limited facility to change the semantics of existing code by adding new code to a compilation (e.g. in a source generator).

A method indicates that it is an by adding one or more attributes. These attributes refer to the source locations of the calls it intercepts.

Any "ordinary method" (i.e. with ) can have its calls intercepted.

In addition to "ordinary" forms and , a call within a conditional access, e.g. of the form can be intercepted. A call whose receiver is a pointer member access, e.g. of the form , can also be intercepted.
