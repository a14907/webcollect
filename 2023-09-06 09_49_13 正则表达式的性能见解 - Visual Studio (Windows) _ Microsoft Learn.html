使用 .NET 5+ 改进正则表达式性能

  • 项目
  • 2023/06/20

本文内容

  1. 原因
  2. 性能见解说明
  3. 如何调查警告

原因

.NET 5 及更高版本中的正则表达式性能已显著改进。 使用大量正则表达式处理的应用程序可以通过切换到 .NET 5+ 来改进性能。

性能见解说明

对 .NET 正则表达式实现的重大改进已在 .NET 5+ 中完成。 有关改进的详细说明，请参阅以下 .NET 博客。

  • .NET 5 中的正则表达式改进
  • .NET 7 中的正则表达式改进

通过使用 BenchmarkDotNet 针对字符串测试正则表达式模式，可以深入了解潜在的性能改进。

如何调查警告

对于 .NET Framework 应用，单击“如何更新”链接以转到 .NET 升级助手，这有助于将项目迁移到 .NET 5+。 另请注意，某些技术在 .NET 5+ 中不可用，需要付出更多的努力进行更新。 有关详细信息，请参阅 .NET Framework 技术在 .NET Core 和 .NET 5+ 上不可用。

对于 .NET Core 应用，此过程更简单；下载并安装要使用的 .NET SDK 版本并更新应用程序的目标框架。

确保应用的部署环境已安装相应的 .NET 运行时版本，以防出现不兼容问题。
