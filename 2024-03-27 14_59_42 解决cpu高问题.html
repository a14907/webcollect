安装工具：
        dotnet tool install -g dotnet-trace && dotnet tool install -g dotnet-counters && export PATH="$PATH:/root/.dotnet/tools"

        确定进程id：
        dotnet-trace ps

        确定运行情况：
        dotnet-counters monitor --refresh-interval 1 -p 1

        抓包：
        dotnet-trace collect -p 1 --providers Microsoft-DotNETCore-SampleProfiler

        复制容器内文件：        
        kubectl cp ux/xx:wdq.dmp wdq.dmp --retries=3
